
## Run Locally

Clone the project

```bash
  git clone https://github.com/Rahul-Maxspike/Spread-Engine.git
```






Create Venv

```bash
 python -m venv venv
```


Install dependencies
```bash
  pip install -r "req.txt"
```


Activate Venv
```bash
 venv\Scripts\Activate
```

Run Dashboard

```bash
python Dashboard.py
  
```
